# دلال - موقع إدارة العقارات

منصة عقارات عربية متكاملة لإدارة وعرض العقارات (أراضي، بيوت، بنايات) مع نظام فلترة وتواصل مباشر.

## 🏠 المميزات

- **عرض العقارات**: عرض جميع العقارات المتوفرة مع صور ومعلومات كاملة
- **نظام فلترة متقدم**: فلترة حسب النوع، الحالة، السعر، والموقع
- **لوحة تحكم الدلال**: إدارة كاملة للعقارات (إضافة، تعديل، حذف)
- **نظام تواصل**: مراسلة مباشرة مع الدلال وعرض أرقام التواصل
- **صلاحيات متقدمة**: تمييز بين الدلال والمستخدم العادي
- **تصميم عصري**: واجهة عربية RTL بالكامل مع تصميم متجاوب

## 📁 هيكل المشروع

```
acakar/
├── index.html          # الصفحة الرئيسية
├── dashboard.html       # لوحة تحكم الدلال
├── property.html        # صفحة تفاصيل العقار
├── contact.html         # صفحة التواصل
├── css/
│   └── style.css        # الأنماط والتصميم
├── js/
│   ├── data.js          # البيانات والتخزين
│   └── app.js           # الوظائف الرئيسية
└── server.js            # خادم التطوير
```

## 🚀 التشغيل

### الطريقة الأولى: باستخدام Node.js
```bash
node server.js
```
ثم افتح http://localhost:8000 في المتصفح

### الطريقة الثانية: باستخدام Python
```bash
python -m http.server 8000
```

### الطريقة الثالثة: استخدام أي خادم ملفات ثابت

## 🔐 بيانات الدخول

- **اسم المستخدم**: admin
- **كلمة المرور**: admin

## 📋 الوظائف

### للمستخدم العادي:
- عرض جميع العقارات
- فلترة العقارات حسب المعايير المختلفة
- مشاهدة تفاصيل العقار
- التواصل مع الدلال

### للدلال (بعد تسجيل الدخول):
- نشر عقارات جديدة
- تعديل معلومات العقارات
- حذف العقارات
- تحديث حالة العقار (متوفر/مباع)
- عرض رسائل التواصل الواردة

## 🎨 التصميم

- **لغة عربية** بالكامل مع دعم RTL
- **تصميم متجاوب** يعمل على جميع الأجهزة
- **واجهة عصرية** باستخدام CSS Grid و Flexbox
- **رسوم متحركة** سلسة وتفاعل احترافي

## 📊 أنواع العقارات

- **قطعة أرض**: أراضي سكنية وتجارية
- **بيت**: بيوت سكنية عائلية
- **بناية**: بنايات سكنية وتجارية

## 🔄 حالات العقارات

- **جاهز**: جاهز للسكن أو الاستخدام
- **قيد الإنشاء**: قيد البناء والتطوير
- **مباع**: تم بيعه (للدلال فقط)

## 📱 التواصل

- نظام مراسلة مدمج
- عرض أرقام الهاتف المباشرة
- تواصل سريع وسهل

## 🛠️ التقنيات المستخدمة

- **HTML5** للهيكل
- **CSS3** للتصميم
- **JavaScript (ES6+)** للبرمجة
- **LocalStorage** للتخزين المحلي
- **Node.js** للخادم (اختياري)

## 📝 ملاحظات

- البيانات مخزنة محلياً في المتصفح
- يمكن رفع الموقع على أي استضافة ثابتة
- التصميم يدعم اللغة العربية بالكامل
- متوافق مع جميع المتصفحات الحديثة
